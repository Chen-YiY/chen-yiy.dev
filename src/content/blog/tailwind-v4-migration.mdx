# Tailwind CSS v4 迁移指南

Tailwind CSS v4 是一次重大更新，引入了全新的配置方式。

## 主要变化

### 1. CSS 优先配置

不再需要 `tailwind.config.js`，直接在 CSS 中配置：

```css
@import "tailwindcss";

@theme {
  --color-primary: #3b82f6;
  --font-sans: 'Inter', sans-serif;
}
```

### 2. 自动内容检测

无需再配置 `content` 数组，Tailwind v4 自动检测模板文件。

### 3. 新的 Vite 插件

使用 `@tailwindcss/vite` 替代 PostCSS 插件：

```javascript
import tailwindcss from '@tailwindcss/vite';

export default {
  plugins: [tailwindcss()],
};
```

## 迁移步骤

1. 移除 `tailwind.config.js`
2. 将配置迁移到 CSS `@theme` 块
3. 更新 `@apply` 和任意值语法
4. 测试所有页面样式

## 注意事项

- `@apply` 语法有细微变化
- 某些插件可能尚未兼容 v4
- 颜色函数语法已更新

## 总结

v4 的 CSS 优先配置让开发体验更流畅，迁移过程总体顺利，推荐尽快升级。
