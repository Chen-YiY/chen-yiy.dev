# i18n 中英文切换 实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 为站点添加中英文 UI 切换功能，涵盖导航、按钮、页面标题等 UI 文本 + About 页面双版本内容。

**Architecture:** 翻译文件 (`en.json` / `zh.json`) 集中管理 UI 文本。Astro 组件中需要翻译的文本用两个 `<span data-lang>` 元素包裹，构建时生成双语文本。React 切换器 (LangToggle) 更新 localStorage 并派发自定义事件，BaseLayout 的内联脚本监听事件并显隐对应语言元素。About 页面用两个 `<section data-lang>` 分别写中英文版本。

**Tech Stack:** Astro 5, React 19, Tailwind CSS v4, localStorage

- [ ] **Step 3: 创建 i18n 工具函数 `src/lib/i18n.ts`**

```typescript
import en from '@/i18n/en.json';
import zh from '@/i18n/zh.json';

const translations = { en, zh } as const;

type Lang = 'en' | 'zh';
type NestedKeyOf<T> = T extends object
  ? { [K in keyof T & string]: T[K] extends object ? `${K}.${NestedKeyOf<T[K]>}` : K }[keyof T & string]
  : never;

export type TranslationKey = NestedKeyOf<typeof en>;

export function getLang(): Lang {
  if (typeof window === 'undefined') return 'zh';
  return (localStorage.getItem('lang') as Lang) || 'zh';
}

export function t(key: TranslationKey, lang?: Lang): string {
  const l = lang || (typeof window !== 'undefined' ? getLang() : 'zh');
  const keys = key.split('.');
  // eslint-disable-next-line @typescript-eslint/no-explicit-any
  let result: any = translations[l];
  for (const k of keys) {
    result = result?.[k];
  }
  return result || key;
}

export function both(key: TranslationKey): { zh: string; en: string } {
  return { zh: t(key, 'zh'), en: t(key, 'en') };
}
```

### Task 2: 创建 LangToggle 切换器组件

**Files:**
- Create: `src/components/LangToggle.tsx`

- [ ] **Step 1: 创建 LangToggle 组件**

```tsx
import { useState, useEffect } from 'react';

type Lang = 'zh' | 'en';

function getStoredLang(): Lang {
  return (localStorage.getItem('lang') as Lang) || 'zh';
}

function applyLang(lang: Lang) {
  document.documentElement.lang = lang === 'zh' ? 'zh-CN' : 'en';
  document.documentElement.setAttribute('data-lang', lang);
  // Toggle visibility of data-lang elements
  document.querySelectorAll<HTMLElement>('[data-lang]').forEach((el) => {
    el.style.display = el.getAttribute('data-lang') === lang ? '' : 'none';
  });
  // Dispatch event for React components
  window.dispatchEvent(new CustomEvent('lang-change', { detail: lang }));
}

export default function LangToggle() {
  const [lang, setLang] = useState<Lang>('zh');

  useEffect(() => {
    const stored = getStoredLang();
    setLang(stored);
    applyLang(stored);
  }, []);

  const toggle = () => {
    const next: Lang = lang === 'zh' ? 'en' : 'zh';
    setLang(next);
    localStorage.setItem('lang', next);
    applyLang(next);
  };

  return (
    <button
      onClick={toggle}
      className="flex items-center gap-1 px-2 py-1 text-xs text-text-secondary hover:text-text-primary transition-colors rounded"
      aria-label={`Language: ${lang === 'zh' ? '中文' : 'English'}`}
      title={lang === 'zh' ? '切换到英文' : 'Switch to Chinese'}
    >
      <svg className="w-3.5 h-3.5" fill="none" viewBox="0 0 24 24" stroke="currentColor" strokeWidth="2">
        <path strokeLinecap="round" strokeLinejoin="round" d="M12 21a9.004 9.004 0 008.716-6.747M12 21a9.004 9.004 0 01-8.716-6.747M12 21c2.485 0 4.5-4.03 4.5-9S14.485 3 12 3m0 18c-2.485 0-4.5-4.03-4.5-9S9.515 3 12 3m0 0a8.997 8.997 0 017.843 4.582M12 3a8.997 8.997 0 00-7.843 4.582m15.686 0A11.953 11.953 0 0112 10.5c-2.998 0-5.74-1.1-7.843-2.918m15.686 0A8.959 8.959 0 0121 12c0 .778-.099 1.533-.284 2.253m0 0A17.919 17.919 0 0112 16.5a17.92 17.92 0 01-8.716-2.247m0 0A9 9 0 013 12c0-1.605.42-3.113 1.157-4.418" />
      </svg>
      <span className={lang === 'zh' ? 'text-text-primary font-medium' : ''}>中</span>
      <span className="text-text-muted">/</span>
      <span className={lang === 'en' ? 'text-text-primary font-medium' : ''}>EN</span>
    </button>
  );
}
```

React 组件中不能使用 `data-lang` 方式（因为它们是客户端水合的），需要通过监听 `lang-change` 事件动态切换文本。

- [ ] **Step 1: 创建 React i18n hook `src/lib/useLang.ts`**

```typescript
import { useState, useEffect } from 'react';
import { t, type TranslationKey } from '@/i18n';

export function useLang() {
  const [lang, setLang] = useState<'zh' | 'en'>(
    (localStorage.getItem('lang') as 'zh' | 'en') || 'zh'
  );

  useEffect(() => {
    const handler = (e: Event) => {
      setLang((e as CustomEvent).detail);
    };
    window.addEventListener('lang-change', handler);
    return () => window.removeEventListener('lang-change', handler);
  }, []);

  return {
    lang,
    t: (key: TranslationKey) => t(key, lang),
  };
```
